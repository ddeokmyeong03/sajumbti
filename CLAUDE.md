# 사주기질 테스트 프로젝트

## 프로젝트 개요
생년월일로 사주 일간을 계산하고, 10개의 상황형 질문으로 MBTI 4기질을 판별해
40가지 사주기질 캐릭터 유형을 보여주는 웹 서비스.
백엔드 없이 순수 프론트엔드(React + Vite)로 구현.

---

## 기술 스택
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **이미지 저장**: html2canvas (결과 카드 공유용)
- **배포**: GitHub Pages
- **개발환경**: GitHub Codespace

---

```
│   └── Result.jsx       # 결과 페이지 + 공유 카드
├── components/

```

## 서비스 흐름
```
시작 페이지
→ 생년월일 입력 (일간 자동 계산)
→ 질문 10개 (기질 판별)
→ 결과 페이지 (캐릭터 + 유형명 + 설명)
→ 결과 카드 공유
```

## 채점 로직
- 각 기질(NF/NT/SJ/SP) 선택 횟수 합산
- 가장 많이 나온 기질 → 최종 기질 결정
- 동점 시 두 번째 기질도 부결과로 표시

```javascript
// 결과 계산 예시
const calcResult = (answers) => {
  const score = { NF: 0, NT: 0, SJ: 0, SP: 0 };
  answers.forEach((type) => score[type]++);
  const sorted = Object.entries(score).sort((a, b) => b[1] - a[1]);
  return { primary: sorted[0][0], secondary: sorted[1][0] };
};
```

## 일간 계산 로직 (만세력 기반)
- 생년월일 입력 → 율리우스 적일 계산 → 일간 도출
- 갑(0) 을(1) 병(2) 정(3) 무(4) 기(5) 경(6) 신(7) 임(8) 계(9)
- `일간index = (율리우스적일 + 기준값) % 10`

```javascript
const getIlgan = (year, month, day) => {
  // 율리우스 적일 계산
  const a = Math.floor((14 - month) / 12);
  const y = year + 4800 - a;
  const m = month + 12 * a - 3;
  const jdn =
    day +
    Math.floor((153 * m + 2) / 5) +
    365 * y +
    Math.floor(y / 4) -
    Math.floor(y / 100) +
    Math.floor(y / 400) -
    32045;

  const ilganNames = ["갑", "을", "병", "정", "무", "기", "경", "신", "임", "계"];
  const ilganKeys = ["거목", "풀꽃", "태양", "촛불", "대산", "대지", "철검", "보석", "대강", "이슬"];
  const index = (jdn + 6) % 10; // 기준값 조정 필요
  return { name: ilganNames[index], key: ilganKeys[index] };
};
```
> ⚠️ 기준값은 실제 만세력과 대조해서 검증 필요
